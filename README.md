# embl-ebi-rest

EMBL-EBI Web Services REST client

## Getting Started
Install the module with: `npm install embl-ebi-rest`

```javascript
var embl_ebi_rest = require('embl-ebi-rest');
embl_ebi_rest.awesome(); // "awesome"
```
